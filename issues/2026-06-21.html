AI 编程代理工具战升温：Claude Code 破译 3500 年古文字，Codex 成为开发者日常主战场

Anthropic 工程师 Boris Cherny 分享了一则令人惊叹的案例：Claude Code 成功破译了 Linear A——一种来自克里特岛、距今 3500 年的未知文字系统。这一「跨界破圈」展示了 AI 编程代理的能力边界正在快速外扩——它不再只是写代码的工具，而是一个可以进入任意知识领域的通用推理引擎。Boris 在推文中写道：「希望这能通过同行评审！」获超 1400 点赞。

与此同时，OpenAI Codex 正在悄悄赢得开发者的心。Thibault Sottiaux（OpenAI Codex & ChatGPT 团队）透露自己在 Mac 上花在 Codex 应用里的时间比其他所有应用加起来还多，并宣布了 Codex 远程/本地无缝切换功能。Vercel CEO Guillermo Rauch 则提出一个大胆判断：「下一个热门编程语言是 Markdown」——在他展示的 eve agent 原型中，整个代理只用两个 markdown 文件定义，一条命令部署。AI 时代的编程范式正在被重新定义。

重塑半导体供应链：Intel CEO 陈立武的豪赌

No Priors · 6月18日

66 岁的陈立武（Lip Bu Tan）在功成名就后选择接手「行业最难的工作」——担任 Intel CEO。他的理由朴素而有力：「这是一家标志性的公司，对半导体生态系统和美国都至关重要。」他上任后的第一步是夯实资产负债表、简化产品线、倾听客户，然后驱动下一代领导力产品。

Jensen Huang 投入 50 亿美元支持 Intel，这笔投资如今已增值至 250 亿。陈立武透露，agentic AI 和推理 CPU 的需求正在爆发式增长，Intel 的 CPU 订单供不应求。访谈中他分享了一条创业铁律：「我投资的 10 家公司中，有 9 家中途改变了商业计划，因为市场变了。」因此他始终相信团队制而非个人英雄主义。

关于芯片制造回流美国，陈立武的判断是：这不仅是商业决策，更是国家安全命题。他的策略是「先爬再走」——保持谦逊，从客户做起，一步步重建 Intel 在先进制程领域的竞争力。
