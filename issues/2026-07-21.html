Stripe 数据与 AI 负责人 Emily Sense 在 MAD Podcast 中描绘了一个正在成形的 Agent 商业世界：AI Agent 将代替人类完成买卖和支付决策，而当前最大的隐患不是「Agent 会超支」，而是 Token 盗窃——超过六分之一的 AI 公司注册来自此类滥用。她透露 Stripe 正在构建全新的「Agent 经济栈」，让每个用户拥有的不是一个 Agent，而是一个负责采购、销售和盈利的 Agent 企业。

Vercel CEO Guillermo Rauch 发表了对网络安全与 AI 关系的深度思考：网络安全是衡量超级智能的最佳基准——发现、修补、逆向和利用漏洞需要超越编程语言和框架的推理能力。Kimi K3 在这一领域的出色表现让他对开源模型的前景充满信心。Box CEO Aaron Levie 则从经济学角度指出，开源模型远强于许多人的预期，限制前沿模型访问只会让美国在竞争中落后。

Every CEO Dan Shipper 宣布了一个里程碑：过去一周，AI 已能自动完成其团队约 70% 的文案编辑工作。「这是第一次达到这个比例，我已经尝试了好几年。」这标志着 AI 在知识工作的又一个关键突破点。与此同时，Anthropic 工程师 Thariq 正在撰写一篇关于 Claude Code 系统提示和 Skills 优化经验的长文，引发了社区的高度关注。

本期播客

Stripe AI 负责人 Emily Sense：AI Agent 如何买卖与支付

The MAD Podcast with Matt Turck · YouTube · 2026-07-09

Emily Sense 认为，当你问普通人是否愿意让 Agent 代为购物时，他们首先担心的是「会不会超支」「会不会买错」「能不能叫停」——这些担忧都是合理的。但更深层的问题是：未来的场景不是 Emily 授权一个 Agent 购物，而是 Emily 拥有一个被分配了经营任务的 Agent，它需要自主采购、销售并创造利润。Token 盗窃是当前 AI 行业最被低估的话题——超过六分之一的 AI 公司注册是这类滥用。

Stripe 正在构建的 Agent 经济栈需要解决三个核心问题：Agent 的身份认证与授权、Token 经济学（按座定价正在瓦解）、以及 Agent 间的交易清算。Emily 预测，12 个月后讨论的主题将是「Agent 企业」——不是人使用工具，而是 Agent 作为经济实体在数字世界中运行。这一愿景正在从概念走向现实。
